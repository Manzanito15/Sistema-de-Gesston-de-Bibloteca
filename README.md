# 📚 Sistema de Gestión de Biblioteca

## 📄 Descripción

Este proyecto consiste en una aplicación de escritorio desarrollada en C# que permite la gestión y administración de una biblioteca de manera eficiente.

## 📌 Funcionalidades del Sistema

* Registrar libros.
* Editar información de los libros.
* Eliminar registros.
* Buscar libros por título, autor o categoría.
* Gestionar préstamos y devoluciones.
* Controlar la disponibilidad de ejemplares.
* Generar reportes de libros y préstamos.

El proyecto fue desarrollado aplicando una arquitectura en capas para una mejor organización, escalabilidad y mantenimiento del código.

---

## 🛠 Tecnologías Utilizadas

* **Lenguaje:** C#
* **Framework:** .NET Framework (Windows Forms)
* **Base de Datos:** SQL Server
* **IDE:** Visual Studio

## ▶ Cómo Ejecutar el Proyecto

1. Clonar el repositorio.
2. Abrir la solución en Visual Studio.
3. Configurar la cadena de conexión en el archivo `App.config`.
